# 콘텐츠 검토 체크리스트

**사용법:** AI 생성 초안 편집 완료 후 발행 전 체크.

---

## 모든 AI 생성 콘텐츠 공통 (필수)

- [ ] 수치·통계·날짜가 원본 소스와 일치하는가?
- [ ] 회사명·인명·직책이 정확한가?
- [ ] 브랜드 보이스 가이드의 금지 표현이 없는가? ("업계 최고", "독보적", "압도적", "노력하겠습니다")
- [ ] 확인되지 않은 사실이 단정적으로 서술되지 않았는가?
- [ ] 법적 리스크 표현 (확약·보증·오해 소지)이 없는가?
- [ ] 담당자가 직접 편집하였는가? (AI 초안 그대로 아님)

---

## 외부 발행 콘텐츠 추가 체크

- [ ] 타 기업·인물에 대한 부적절한 비교·언급이 없는가?
- [ ] 경쟁사 정보가 포함되어 있지 않은가?
- [ ] 아직 공개되지 않은 내부 정보가 없는가?
- [ ] 팀장 최종 승인 완료

---

## 채널별 추가 체크

### SNS (LinkedIn / Instagram)
- [ ] 해시태그 적절성 확인
- [ ] 이모지 사용이 브랜드 보이스 가이드 기준 이내
- [ ] 링크가 있는 경우 정상 작동 확인
- [ ] 이미지/영상 저작권 확인

### 보도자료
- [ ] 제목 30자 이내
- [ ] 리드(2문장)에 핵심 전달 여부
- [ ] 인용구 대상자 직책·이름 최종 확인
- [ ] 문의처 정보 정확 여부

### CEO 연설문·메시지
- [ ] CEO 보이스 프로파일과 어조 일치 여부
- [ ] 발표 시간 내 분량인지 확인 (1분 ≈ 200자)
- [ ] 발음하기 어려운 표현 없는지 소리 내어 확인

### 위기 대응
- [ ] 확인된 사실만 포함 (추측 없음)
- [ ] 등급별 승인자 확인 완료
- [ ] 모든 채널 메시지 일관성 확인
