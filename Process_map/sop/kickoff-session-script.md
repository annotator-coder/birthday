## 섹션 1: 왜 지금 AI인가 (5분)

**스크립트:**

> "우리 팀은 지금 변화의 기로에 있습니다. 홍보 업무에 AI를 도입하는 건 선택이 아니라 필수가 됐어요.
>
> 우리가 AI를 쓰는 이유는 단순히 '빠르게'가 아닙니다. 더 좋은 콘텐츠를, 더 많은 채널에, 지금 인력으로 만들기 위해서입니다. 오늘부터 우리 팀의 모든 업무에 AI가 함께합니다.
>
> 목표는 명확합니다: 3개월 내에 팀 모두가 AI를 자신의 업무에 자연스럽게 쓰는 것. 오늘이 그 시작입니다."

**슬라이드 포인트:** 팀 대시보드의 10개 업무 도메인 → 각 도메인에 AI가 투입되는 모습

---

## 섹션 2: 라이브 데모 3가지 (10분)

### 데모 1 — 보도자료 초안 (3분 30초)

**시나리오:** "방금 MOU를 체결했는데 내일까지 보도자료를 써야 해요. 예전 방식 vs AI 방식 비교해 볼게요."

**진행 순서:**
1. `prompts/pr/press-release-templates.md` 파일 열기 (15초)
2. 유형 A 프롬프트 복사 → Claude 열기 (15초)
3. [ANNOUNCEMENT]에 "GS칼텍스-○○사 수소 협력 MOU 체결" 입력 등 변수 채우기 (1분)
4. 생성 결과 화면 보여주기 (1분)
5. "이 초안, 몇 % 쓸 수 있을 것 같아요?" 팀원들에게 질문 (30초)

### 데모 2 — Instagram 캡션 (3분)

**시나리오:** "SNS 팀원이 오늘 포스팅해야 하는데 아이디어가 안 떠올라요."

**진행 순서:**
1. `prompts/brand-voice/sns-caption-prompts.md` → Instagram 프롬프트 복사 (30초)
2. 최근 미디어허브 기사 1건을 [TOPIC]으로 입력 (1분)
3. 2가지 버전 생성 결과 공유 (1분)
4. "어떤 버전이 더 GS칼텍스답다고 느껴져요?" 질문 (30초)

### 데모 3 — 경영진 보고서 (3분 30초)

**시나리오:** "분기 성과 보고서 쓸 때 항상 시간이 많이 걸렸죠. 이제는 이렇게 합니다."

**진행 순서:**
1. `prompts/shared/stakeholder-report-prompt.md` 열기 (30초)
2. 이번 분기 대표 성과 3가지 입력해서 초안 생성 (2분)
3. "30초 내에 핵심 파악 가능한지" 확인 (1분)

## 섹션 3: 나의 첫 번째 AI 과제 (10분)

**진행 방법:**
1. 각 팀원이 포스트잇(또는 채팅창)에 적는다: "나는 이번 달 [업무명]에 AI를 써보겠다."
2. 팀장이 각자 발표 받음 (30초씩)
3. 과제 기준 확인:
   - 이번 달 안에 실제로 해볼 수 있는 것
   - 결과를 프롬프트 뱅크에 기록할 수 있는 것

**예시 과제 (참고용):**
- SNS 팀: "다음 주 LinkedIn 포스팅 3개를 AI로 초안 생성"
- PR 팀: "다음 보도자료 작성할 때 유형 A 템플릿 활용"
- Internal Comm: "다음 G식노트 스크립트 초안을 AI로"
- PI: "CEO 연설문 아카이브 수집 + Claude Projects 설정"
