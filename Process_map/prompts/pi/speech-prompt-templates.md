## Claude Projects CEO 보이스 설정 가이드

### Step 1: 연설문 아카이브 수집 (최소 10건)
아래 문서를 수집하여 `Process_map/knowledge-base/ceo-speeches/` 폴더에 저장 (비공개):
- 신년사 (최근 3년)
- 창립기념사
- EDP 메시지
- 배구단 관련 CEO 메시지
- 기타 공식 발언문·인터뷰 텍스트

### Step 2: Claude Projects 생성
1. Claude.ai → Projects → "GS칼텍스 CEO 보이스" 프로젝트 생성
2. 수집한 연설문 파일 업로드
3. 아래 시스템 프롬프트(지시사항) 입력:

```
당신은 GS칼텍스 CEO의 연설문과 공식 메시지를 작성하는 전문 작가입니다.
업로드된 과거 연설문을 완전히 학습하여 CEO의 고유한 어법·가치관·
표현 방식을 재현해야 합니다.

분석 후 아래 항목을 이 프롬프트에 추가하세요:
- CEO 자주 쓰는 표현 5개: [문서 분석 후 작성]
- 핵심 가치 키워드 5개: [문서 분석 후 작성]
- 피하는 표현: [문서 분석 후 작성]
- 문장 길이 특징: [문서 분석 후 작성]
- 특징적 구성 패턴: [문서 분석 후 작성]
```

### Step 3: 분석 프롬프트 (프로젝트 생성 직후 실행)
업로드 완료 후 아래 프롬프트를 입력하여 보이스 프로파일을 추출:

```
업로드된 연설문들을 분석해서 CEO 보이스 프로파일을 작성해줘.

1. 자주 쓰는 표현·단어 Top 10
2. 핵심 가치 키워드 (연설에서 반복되는 주제)
3. 문장 구조 특징 (길이, 수사법, 리듬)
4. 피하는 표현이나 스타일
5. 도입부·결론부 패턴
6. 연설 유형별 (격려/발표/기념/위기) 톤 차이

분석 결과를 "CEO 보이스 원칙 요약 (5줄)" 형식으로도 정리해줘.
```

## 연설문 생성 프롬프트 (Claude Projects에서 사용)

### 행사 연설문

```
아래 조건으로 연설문을 작성해주세요.

[행사 정보]
- 행사명: [EVENT_NAME]
- 날짜/장소: [DATE_VENUE]
- 청중: [AUDIENCE]
- 예상 발표 시간: [DURATION]분
- 핵심 메시지: [KEY_MESSAGE]
- 포함할 사실/수치: [FACTS_DATA]
- 특별히 언급할 인물/사건: [SPECIAL_MENTION] (없으면 "없음")

[DURATION]분 발표 기준 분량으로 작성하고,
청중에게 전달할 핵심 한 문장(headline sentence)도 별도로 제안해줘.
```

### 짧은 CEO 메시지 (사내공지·영상·뉴스레터용)

```
아래 상황에 맞는 CEO 메시지를 작성해주세요.

[상황]
- 목적: [PURPOSE] (격려/발표/감사/위기 등)
- 대상: [AUDIENCE]
- 핵심 전달 사항: [CORE_MESSAGE]
- 길이: [SHORT/MEDIUM/LONG] (100자/300자/500자 기준)

메시지 2가지 버전을 작성해줘: 공식 버전과 좀 더 따뜻한 버전.
```

### 연설문 피드백·수정 프롬프트

```
아래 연설문 초안을 CEO 보이스에 맞게 검토하고 수정해줘.

[초안]
[DRAFT — 초안 붙여넣기]

검토 요청:
1. CEO 보이스에서 벗어난 표현 지적 (이유 포함)
2. 수정 버전 제안
3. 전체적으로 CEO답게 느껴지는지 0~10점 평가 (이유 포함)
```

## 성공 기준
"이 정도면 CEO에게 초안으로 보여줄 수 있다" 수준.
Claude Projects에 연설문 10건 이상 업로드 후 분석 프롬프트 실행 → 보이스 프로파일 문서화 → 팀장 승인.
