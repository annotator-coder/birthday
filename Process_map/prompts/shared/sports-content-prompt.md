## 경기 결과 SNS 콘텐츠 패키지 프롬프트

### 사용법
경기 직후 사용. `[SPORT_TEAM]`, `[RESULT]`, `[PLAYER_HIGHLIGHTS]`, `[NEXT_SCHEDULE]` 교체.

### 프롬프트

```
GS칼텍스 스포츠팀 경기 결과를 SNS 콘텐츠로 만들어주세요.

[경기 정보]
- 종목/팀: [SPORT_TEAM] (예: GS칼텍스 배구단, GS칼텍스 바둑팀)
- 경기 결과: [RESULT] (승/패, 스코어)
- 주요 선수 활약: [PLAYER_HIGHLIGHTS]
- 다음 일정: [NEXT_SCHEDULE]

아래 3가지를 작성해줘:
1. LinkedIn 포스팅 (자긍심 중심, 150~250자)
2. Instagram 캡션 + 해시태그 5~7개
3. 팬·임직원 응원 댓글 답글 예시 2가지

톤 기준:
- 승리 시: 자긍심 고취, 선수 칭찬, 팬들에게 감사
- 패배 시: 응원·격려, 다음 경기 기대감, 선수 노력 인정
```

## 시즌 개막·종료 캠페인 메시지 프롬프트

### 프롬프트

```
스포츠팀 시즌 [개막/종료] 공식 메시지와 SNS 콘텐츠를 작성해주세요.

[시즌 정보]
- 종목/팀: [SPORT_TEAM]
- 시즌: [SEASON] (예: 2025-26 V리그)
- 주요 목표/성과: [SEASON_HIGHLIGHT]
- 선수단 특이사항: [TEAM_NEWS]

작성 요청:
1. CEO/단장 공식 메시지 (300자)
2. SNS 시즌 개막/종료 포스팅 (LinkedIn + Instagram)
3. 응원 캠페인 해시태그 제안 3가지
```
