## 경영진 보고서 초안 프롬프트

### 사용법
`[AUDIENCE]`, `[PURPOSE]`, `[KEY_RESULTS]`, `[ISSUES]`, `[DECISION_NEEDED]`, `[DATA]` 교체 후 Claude에 입력.

### 프롬프트

```
당신은 GS칼텍스 홍보부문 팀장의 보고서 작성을 돕는 어시스턴트입니다.

[보고 정보]
- 보고 대상: [AUDIENCE] (예: 홍보실장, CMO)
- 보고 목적: [PURPOSE]
- 핵심 성과/현황: [KEY_RESULTS]
- 이슈/리스크: [ISSUES]
- 요청 사항/의사결정 필요 항목: [DECISION_NEEDED]
- 첨부할 데이터: [DATA]

[형식 조건]
- 결론 먼저, 근거 나중 (Pyramid Structure)
- 총 A4 1~2장
- 각 섹션: 요약 bullet → 상세 설명 구조

보고서 초안을 작성해주세요. 보고 대상이 30초 내에 핵심을 파악할 수 있어야 합니다.
```

## VM 성과 취합 자동화 프롬프트

### 사용법
분기말 팀원별 성과 데이터를 취합할 때 사용. `[MEMBER_DATA]` 부분에 각 팀원 성과를 붙여넣기.

### 프롬프트

```
아래 팀원별 성과 데이터를 취합하여 분기 성과 보고서 형식으로 정리해주세요.

[팀원별 성과 데이터]
[MEMBER_DATA — 각 팀원의 업무 항목과 성과 붙여넣기]

[출력 형식]
1. 전체 요약 (3줄)
2. VM 전략과업 진행 현황 (과업별 상태: 완료/진행중/지연)
3. 카테고리별 성과 집계 (내부소통/PR/CSR/콘텐츠/SNS 등)
4. 주요 성과 하이라이트 5개
5. 다음 분기 예정 사항
```

---

## 성공 기준
보고 대상이 30초 내에 핵심을 파악할 수 있는 구조인지 확인.
