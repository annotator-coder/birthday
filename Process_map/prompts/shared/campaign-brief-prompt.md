## 캠페인 기획안 초안 프롬프트

### 사용법
`[OBJECTIVE]`, `[TARGET]`, `[KEY_MESSAGE]`, `[BUDGET_RANGE]`, `[DURATION]`, `[CHANNELS]`, `[REFERENCE_AVOID]` 교체.

### 프롬프트

```
당신은 GS칼텍스 홍보팀 캠페인 기획자입니다.

[브랜드 보이스]
GS칼텍스는 국내 대표 에너지 기업. 전문적이면서 친근한 톤.
사실과 수치 중심. 금지: "업계 최고", "독보적", "압도적".

[캠페인 기본 정보]
- 캠페인 목적: [OBJECTIVE]
- 타겟 오디언스: [TARGET]
- 핵심 메시지: [KEY_MESSAGE]
- 예산 수준: [BUDGET_RANGE]
- 기간: [DURATION]
- 활용 채널: [CHANNELS]
- 참고 레퍼런스 또는 금지 방향: [REFERENCE_AVOID]

캠페인 기획안 초안을 작성해주세요:
1. 캠페인 콘셉트 3가지 옵션 (각 한 줄 + 장단점)
2. 추천 옵션 상세 기획:
   - 핵심 크리에이티브 방향
   - 채널별 콘텐츠 계획
   - 주차별 실행 타임라인
3. 예상 KPI (채널별)
4. 리스크 및 유의사항
```

## 캠페인 성과 중간 점검 프롬프트

### 프롬프트

```
진행 중인 캠페인의 중간 성과를 분석하고 최적화 방향을 제안해주세요.

[캠페인 정보]
- 캠페인명/목적: [CAMPAIGN_INFO]
- 진행 기간: [PERIOD]
- 현재까지 성과 데이터: [PERFORMANCE_DATA]
- 목표 KPI: [TARGET_KPI]

분석 요청:
1. 현재 성과 vs 목표 갭 분석
2. 잘 되고 있는 요소 / 개선 필요 요소
3. 남은 기간 최적화 액션 3가지
```
