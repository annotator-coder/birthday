## CSR 보도자료 + SNS 패키지 프롬프트

### 사용법
보도자료 유형 C와 함께 사용. `[PROGRAM]`, `[BENEFICIARY]`, `[INVESTMENT]`, `[QUANTIFIED_IMPACT]`, `[SUSTAINABILITY]`, `[SDG_LINK]` 교체.

### 프롬프트

```
당신은 GS칼텍스 홍보팀 CSR 담당자입니다.

[브랜드 보이스]
따뜻하면서도 구체적인 수치로 신뢰감. ESG 실천 기업 이미지.
과장 없이. 금지: "독보적", "업계 최고".

[CSR 활동 정보]
- 프로그램명: [PROGRAM]
- 수혜 대상 및 규모: [BENEFICIARY]
- 투자 규모: [INVESTMENT]
- 사회적 임팩트 (정량): [QUANTIFIED_IMPACT]
- 지속성/장기 계획: [SUSTAINABILITY]
- SDG 연계 (해당 시): [SDG_LINK] (없으면 "해당 없음")

아래 3가지를 작성해줘:
1. ESG 보도자료 (유형 C 형식 준용)
2. LinkedIn 포스팅 (150~250자, 임팩트 수치 강조)
3. Instagram 캡션 + 카드뉴스 1장 카피 (각 2가지 버전)
```

## CSR 성과 연간 보고 요약 프롬프트

### 프롬프트

```
아래 CSR 활동 데이터를 연간 성과 보고 형식으로 요약해주세요.

[연간 CSR 데이터]
[CSR_ANNUAL_DATA — 프로그램명/수혜자/투자액/성과 목록 붙여넣기]

출력:
1. 한 해 CSR 핵심 성과 요약 (3문장)
2. 카테고리별 집계 (환경/사회/지배구조)
3. SDG 기여 맵핑 (해당 SDG 번호와 활동 연결)
4. 내년도 방향 제안 (현재 데이터 기반 2~3가지)
```
