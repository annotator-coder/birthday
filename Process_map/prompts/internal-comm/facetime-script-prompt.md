# Face+Time 스크립트 생성 프롬프트

## 시리즈 개요
- 형식: GS칼텍스 임직원 인터뷰 시리즈, 사람의 이야기에 집중
- 어조: 진솔하고 따뜻한, 조직 공감대 형성

---

## 파트 1: 인터뷰 질문지 생성 프롬프트

### 사용법
`[NAME_TITLE]`, `[ROLE]`, `[EPISODE_ANGLE]`, `[RECENT_EXPERIENCE]` 교체 후 Claude에 입력.

### 프롬프트

```
당신은 GS칼텍스 Face+Time 시리즈 인터뷰어입니다.
이 시리즈는 임직원의 진솔한 이야기를 통해 조직 공감대를 만듭니다.

[인터뷰 대상자 정보]
- 이름/직급: [NAME_TITLE]
- 담당 업무: [ROLE]
- 이번 에피소드 주제/앵글: [EPISODE_ANGLE]
- 최근 주요 경험/성과: [RECENT_EXPERIENCE]

인터뷰 질문 10개를 아래 구성으로 작성해주세요:
- 워밍업 질문 2개 (가벼운 이야기로 시작)
- 핵심 질문 6개 (에피소드 앵글 중심)
- 마무리 질문 2개 (메시지, 동료에게 한마디)

각 질문 옆에 의도와 예상 답변 방향도 한 줄씩 설명해줘.
```

---

## 파트 2: 인터뷰 녹취 → 편집 스크립트 변환 프롬프트

### 사용법
인터뷰 녹취 후 사용. `[TRANSCRIPT]`, `[DURATION]`, `[KEY_MESSAGE]` 교체 후 입력.

### 프롬프트

```
아래 인터뷰 녹취록을 Face+Time 시리즈 편집 스크립트로 변환해주세요.

[녹취록]
[TRANSCRIPT — 녹취 텍스트 붙여넣기]

[편집 방향]
- 총 길이: [DURATION]분
- 핵심 메시지: [KEY_MESSAGE]

편집 스크립트를 아래 형식으로 작성해주세요:
1. 사용할 인터뷰 발언 선별 (텍스트 인용, 예상 분량 표기)
2. 나레이션/자막이 필요한 부분 및 내용
3. B-roll 제안 (어떤 장면을 삽입하면 좋을지)
4. 오프닝 훅 (15초 이내, 시청자를 붙잡는 발언이나 장면)
5. 클로징 문구
```

---

## 검증 기준
- 질문지: 인터뷰 대상자가 자연스럽게 답할 수 있는 구성인지 확인
- 편집 스크립트: 실제 편집자가 참고할 수 있는 구체성인지 확인
