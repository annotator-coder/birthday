# G식노트 스크립트 생성 프롬프트

## 시리즈 개요
- 형식: 사내 지식·인사이트 공유 영상 시리즈
- 어조: 친근한 강의체 — 전문 내용을 쉽고 흥미롭게
- 구성 패턴: 훅(문제 제기) → 본론(3가지 핵심) → 실천 포인트 → 마무리
- 대상: GS칼텍스 임직원 전체

---

## 스크립트 생성 프롬프트

### 사용법
`[TOPIC]`, `[KEY_MESSAGE_1~3]`, `[DURATION]`, `[REFERENCE_MATERIAL]` 교체 후 Claude에 입력.

### 프롬프트

```
당신은 GS칼텍스 G식노트 시리즈 스크립트 작가입니다.
이 시리즈는 임직원이 알아야 할 지식·인사이트를 친근한 강의체로 전달합니다.

[에피소드 정보]
- 주제: [TOPIC]
- 핵심 메시지 3가지: [KEY_MESSAGE_1], [KEY_MESSAGE_2], [KEY_MESSAGE_3]
- 대상 시청자: GS칼텍스 임직원 전체
- 목표 길이: [DURATION]분
- 참고 자료: [REFERENCE_MATERIAL]

[시리즈 구성 패턴]
1. 훅 (0:00~0:30): 오늘 다룰 내용이 왜 중요한지 문제 제기
2. 본론 (0:30~종료 2분 전): 핵심 메시지 3가지를 순서대로
3. 실천 포인트 (종료 1분 전): 오늘 바로 해볼 수 있는 것 1~2가지
4. 마무리 (마지막 30초): 핵심 요약 + 다음 에피소드 예고

스크립트를 아래 형식으로 작성해주세요:
- 나레이터 대사 / [화면 지시: 어떤 화면이나 자막이 필요한지]
- 각 섹션의 예상 소요 시간 표기
- 시청자 참여를 유도하는 질문 또는 인터랙션 포인트 1개 포함 (본론 중간)
```

---

## 검증 기준
생성된 스크립트가 기존 에피소드의 톤·구성·길이와 유사한지 담당자(이현상 책임)와 함께 확인.
