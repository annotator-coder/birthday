# PR2팀 AI 지식베이스

**목적:** 팀원들이 발견한 좋은 프롬프트, 실전 사례, 학습 내용을 기록·공유하는 공간

---

## 폴더 구조

```
knowledge-base/
├── README.md               ← 이 파일
├── prompt-bank.md          ← 검증된 프롬프트 모음 (항상 업데이트)
├── case-studies/           ← 실제 적용 사례 (Before/After)
│   └── [YYYY-MM-DD]-[도메인]-[주제].md
└── ceo-speeches/           ← CEO 연설문 아카이브 (비공개, .gitignore)
```

---

## 기록하는 방법

### 새로운 좋은 프롬프트를 발견했을 때
1. `prompt-bank.md` 열기
2. 해당 도메인 섹션에 아래 형식으로 추가:
   ```
   **상황:** [어떤 업무에 사용했는지]
   **평점:** ★★★★☆
   **프롬프트:** [사용한 프롬프트 전문]
   **결과 메모:** [어떤 점이 좋았는지, 다음에 개선할 점]
   ```
3. 팀 채널에 공유 ("이 프롬프트 써봤더니 좋더라" + 링크)

### 실제 사례를 기록할 때
`case-studies/` 폴더에 파일 생성 (예: `2026-05-20-SNS-에너지전환포스팅.md`):
- Before: 기존에 어떻게 했는지
- After: AI 활용 후 어떻게 달라졌는지
- 소요 시간 비교
- 배운 점
